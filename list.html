Data Structures Tutorial

List
A List is the simplest abstract data type, simply an ordered collection of values. This can be implemented in several ways, such as using an array:

int intList[] = {0,1,2,3,4};


Another way is to create a "Linked List"- creating a sequence of Nodes(values)which point to the next value:

public class Node{
  int value;
  public Node(int value){
    this.value = value;
  }
  public Node(int value, Node next){
    this.value = value;
    this.next = next;
  }
  public void setNext( Node next){
    this.next = next;
  }
}


In this implementation, the first Node in our sequence is called the "head" and the last Node the "tail". To keep track of the List, we simply need to keep track of the head then traverse the "next" pointer of each Node to access the rest of the list. The tail Node has no next Node, therefore we assign null. An example of a linked list:

Node head = new Node(1);
Node second = new Node(3);
Node tail = new Node(5);
head.setNext(second);
second.setNext(tail);

Node currentNode;
currentNode=head;
while (currentNode.next != null)
{
  System.out.println(value + " ");
  currentNode = currentNode.next;
}

This example will print the sequence "1 3 5". Varients of the linked list are very popular:
Doubly linked list- Same as the example above, but with a "previous" Node pointer. This allows traversal of the list in either direction.
ArrayList- An arraylist is a linked list with indexies. This costs more internal storage, but allows us randomized access.
