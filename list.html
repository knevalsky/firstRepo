Data Structures Tutorial

List
A List is the simplest abstract data type, simply an ordered collection of values. This can be implemented in several ways, such as using an array:

int intList[] = {0,1,2,3,4};


Another way is to create a "Linked List"- creating a sequence of Nodes(values)which point to the next value:

public class Node{
  int value;
  Node next;
  public Node(int value, Node next){
    this.value = value;
    this.next = next;
  }
}


In this implementation, the first Node is called the "head" and the last Node the "tail". To keep track of the List, we simply need to keep track of the head then traverse the "next" pointer of each Node to access the rest of the list. The tail Node has no next Node, therefore we assign null.
